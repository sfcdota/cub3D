# Basic raycasting engine implementation of cub3D task

System requirements: MacOS

To run:

`git clone https://github.com/sfcdota/cub3D`

Go to root dir and:

`make`

Then:

`./cub3D {*.cub}`

Where *.cub the name of config file in the root dir.

More info in [cub3D.pdf](https://github.com/sfcdota/cub3D/blob/master/cub3D.pdf)
